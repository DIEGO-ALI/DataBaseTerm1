Actividad de Consulta a Base de Datos

El objetivo de esta página es poder compartir el proceso que se llevó a cabo en la actividad de consulta a la base de datos. La temática elegida fue el desarrollo web, cada uno de los tres integrantes del grupo hicieron consultas individuales sobre la temática por lo que se dividieron las consultas de cada quien en pequeñas subpginas que se podrán consultar mediante el menú de a continuación o el menú de la parte superior.
